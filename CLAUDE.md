# Modern Thesaurus

## Project Overview

A full-stack crowd-sourced modern thesaurus web app where users can define, vote on, and discuss contemporary words, slang, and modern terminology. Captures the evolving language of today.

## Features

- **Crowd-Sourced Definitions**: Community creates and refines definitions
- **Voting System**: Upvote/downvote definitions to surface the best ones
- **Discussion Threads**: Debate and discuss word meanings
- **Modern Slang**: Focus on contemporary language and slang
- **Search & Browse**: Easy discovery of words and definitions
- **User Contributions**: Register to submit and vote

## Architecture

### Frontend
- React-based single-page application
- Real-time voting and discussion
- Clean, intuitive UI for word discovery

### Backend
- Node.js/Express server
- PostgreSQL or similar database
- User authentication for voting
- API for word and definition management

## Tech Stack

- **Frontend**: React + TypeScript
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **Deployment**: GitHub Pages (frontend) + hosted backend

## How to Use

1. Browse modern words and their definitions
2. Search for slang or contemporary terminology
3. Vote on definitions (create account to vote)
4. Submit your own definitions for new words
5. Participate in discussions about word meanings

## Database Schema

- **Words**: Word entries with metadata
- **Definitions**: User-submitted definitions with timestamps
- **Votes**: Upvotes/downvotes with user tracking
- **Comments**: Discussion threads on definitions
- **Users**: Account information for contributors
